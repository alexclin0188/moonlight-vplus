## 6.1 StreamActivity 返回键

**文件**：`StreamActivity.kt`

```kotlin
// 防抖
private var lastBackPressTime = 0L
private val backPressDebounceMs = 300L

// 面板状态（需要从 StreamOverlay 通过回调或 ViewModel 共享）
private var panelState = PanelState.HIDDEN
private var detailPage: DetailPage? = null

fun onPanelStateChanged(state: PanelState, detail: DetailPage?) {
    panelState = state
    detailPage = detail
}

override fun onBackPressed() {
    val now = SystemClock.elapsedRealtime()
    if (now - lastBackPressTime < backPressDebounceMs) {
        return  // 防抖：忽略过快操作
    }
    lastBackPressTime = now

    when {
        // 面板未展开 → 展开操作面板
        panelState == PanelState.HIDDEN -> {
            onPanelStateChanged(PanelState.SUB_PANEL, null)
            // 通知 Compose 层展开面板
        }
        // 详情页可见 → 返回主列表
        panelState == PanelState.SUB_PANEL && detailPage != null -> {
            onPanelStateChanged(PanelState.SUB_PANEL, null)
        }
        // 子面板可见 → 收起至竖向窄条
        panelState == PanelState.SUB_PANEL || panelState == PanelState.KEYBOARD_PANEL -> {
            onPanelStateChanged(PanelState.VERTICAL_BAR, null)
        }
        // 仅竖向窄条 → 收起
        panelState == PanelState.VERTICAL_BAR -> {
            onPanelStateChanged(PanelState.HIDDEN, null)
        }
        // 不应该走到这里
        else -> super.onBackPressed()
    }
}
```

### Compose 层集成

在 `StreamOverlay.kt` 中，将 `panelState` 和 `detailPage` 提升为可由 Activity 修改的状态：

```kotlin
// 方案：使用 CompositionLocal 或 callback
@Composable
fun StreamOverlay(
    engine: StreamEngine,
    externalPanelState: PanelState,       // 由 Activity 传入
    externalDetailPage: DetailPage?,      // 由 Activity 传入
    onStateChanged: (PanelState, DetailPage?) -> Unit  // 通知 Activity
) {
    var panelState by remember { mutableStateOf(externalPanelState) }
    var detailPage by remember { mutableStateOf(externalDetailPage) }

    // 当 externalPanelState 变化时同步
    LaunchedEffect(externalPanelState) {
        panelState = externalPanelState
    }
    LaunchedEffect(externalDetailPage) {
        detailPage = externalDetailPage
    }

    // 状态变化时通知 Activity
    fun updateState(state: PanelState, detail: DetailPage?) {
        panelState = state
        detailPage = detail
        onStateChanged(state, detail)
    }
    // ... 其余代码中，所有修改 panelState/detailPage 的地方改为调用 updateState()
}
```

## 6.2 收尾工作

### 主题适配

所有面板 Composable 使用 `MoonLinkTheme` 的颜色 tokens：
- `primary = Color(0xFF00B4D8)` — 青色主色
- `onPrimary = Color(0xFF003544)`
- `primaryContainer = Color(0xFF004D5C)`
- `secondary = Color(0xFFFF6B35)` — 橙红辅色
- `background = Color(0xFF0D0B1A)` — 深紫黑
- `surface = Color(0xFF1C1B2E)` — 暗紫灰
- `surfaceVariant = Color(0xFF2D2A3E)` — 浅紫灰
- `onBackground/onSurface = Color(0xFFE6E1E5)` — 浅灰文字

### 空状态处理

- 超级指令为空时（`cmdList` 为 null 或空数组）：显示"暂无可用指令，请在 Sunshine 基地版中添加自定义命令"
- 无自定义按键时：显示"暂无自定义按键，点击 + 添加"
- 无显示器列表时：显示"无法获取显示器信息"

### 圆形/间距规范

- 卡片/面板圆角：16dp
- 按钮圆角：20dp（胶囊形）
- 列表项圆角：8dp
- 内边距：12-16dp
- 图标大小：20-24dp

### 横竖屏适配

面板使用 `fillMaxHeight()` 自动适应竖屏和横屏高度变化。
子面板宽度在横屏时可能更窄，使用 `coerceIn(280.dp, 400.dp)` 限制。

## 6.3 产出验证

1. 连续快速按返回键（< 300ms 间隔）不触发多次面板切换
2. 各级面板展开时按返回键逐步收起：
   - 详情 → 主列表
   - 子面板 → 竖向窄条
   - 竖向窄条 → 悬浮按钮
3. 仅悬浮按钮可见时按返回键 → 展开操作面板（竖向窄条 + 操作子面板）
4. 面板全部收起后再按返回 → `super.onBackPressed()`（退出串流）
5. 画中画模式下返回键行为正常
