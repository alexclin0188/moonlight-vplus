# 原始需求

参考 UU 远程的键盘界面实现，点击窄面板上的键盘之后，像 UU 远程一样弹出一个满屏宽度横条，上面有三个选项"输入法"/"快捷键"/"虚拟键盘"：
- 输入法：呼出系统输入法全键盘
- 快捷键：默认提供和自定义的快捷键
- 虚拟键盘：用旧代码的虚拟键盘（复制其代码略作改动）
- 默认选择输入法

## UU 远程截图分析

### 截图 1：输入法标签
- 顶部横条三个标签：输入法/快捷键/电脑键盘
- 输入法选中时显示系统中文 QWERTY 键盘
- 右侧有关闭按钮（X）

### 截图 2：快捷键标签
- 横向标签栏，快捷键选中（蓝色文字+蓝色下划线）
- 左上角"编辑"按钮（蓝色铅笔图标）
- 2×6 网格按钮：Caps Lock/Ctrl+C/Ctrl+V/Ctrl+X/Ctrl+A/Ctrl+Z/Ctrl+S/Win/Win+D/Win+Tab/Win+L
- 每个按钮显示快捷键名+中文描述
- 深色主题

### 截图 3：电脑键盘标签
- 完整 QWERTY 虚拟键盘
- 顶部修饰键行：组合键模式复选框 + Ctrl/Shift/Alt/Windows
- 标准键盘布局，深色主题
- 底部分页指示器（两个点）
